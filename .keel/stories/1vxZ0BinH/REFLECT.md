## Observations

A `CommandGuidance` intermediary model made the guidance pathway explicit and removed duplicate payload-construction logic.
Refactoring `next` to map to command decisions first and render payload second preserved semantics while reducing drift risk between branches.
Focused tests around helper rendering and JSON command mapping gave stronger contract guarantees than broad behavioral assertions.
